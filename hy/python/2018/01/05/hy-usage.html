Hy Usage

·

January 05, 2018

package hy module:

  1. build a normal python package
  2. write .hy file, and add import hy in toplevel or just __init__.py (to enable compile pyc using hy)
  3. write package data in setup on setup.py 
    setup = (
    package_data={
            'minghu6.text': ['*.hy', '__pycache__/*'],
        },
)
